# Future REST API & Axios Integration

This directory is designated for future integration with the backend REST APIs. 

## Structure Outline

Once backend development begins:
- Create `client.ts` to initialize the Axios client instance with standard baseURL, timeout configuration, and interceptors (for automatic JWT token injection and error handling).
- Create endpoints mapping directories (e.g. `/employees`, `/devices`).

## Planned Libraries
- **Axios** will be used for client requests.
